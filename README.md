# Heart Rate Analysis Project

This is a simple Python project for analyzing and visualizing heart rate data.

## What it does
- Calculates average heart rate
- Finds maximum and minimum values
- Plots heart rate changes over time

## Why this project?
This project is made to practice:
- Python basics
- NumPy for data analysis
- Matplotlib for visualization
- Working with simple biomedical signals

## Technologies used
- Python
- NumPy
- Matplotlib
